# Support

Use the repository issue templates for project support:

- bug reports for reproducible defects;
- feature requests for new contract, adapter or CLI capabilities;
- adapter gap reports when a platform cannot preserve a ContractForge semantic
  behavior.

For security issues, do not open a public issue. See [SECURITY.md](SECURITY.md).

For usage questions, include:

- package and adapter versions;
- Python version;
- target platform;
- minimal contract snippet;
- command executed;
- redacted error output.
