# Changelog

All notable changes to `contractforge-fabric` are documented in this file.

## [0.2.0] - 2026-06-19

### Added

- Added release metadata for the current Fabric adapter baseline covering
  Lakehouse notebook execution, OneLake artifact handling, shortcuts, file
  readers, quality/evidence, governance review/apply helpers and project
  deployment promotion.
- Documented the capacity-constrained execution boundary as an operational
  platform concern rather than an adapter workaround.

### Changed

- Aligned the adapter dependency range with `contractforge-core` 0.2.x.
- Promoted the documented `fabric_lakehouse` surface to the current stable
  supported release boundary.

## [0.1.0] - 2026-06-08

- Added the initial render-only Fabric Lakehouse adapter package.
- Added conservative capabilities, source-support classification, review
  artifacts and CLI commands.
